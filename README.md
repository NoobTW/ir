# Homework of Information Retrieval (IMF501)

## Requirements

- Python 2.7

## Midterm Homework

`mid_1.py` will generate `weight1.txt`, and `mid_2.py` will generate `weight.txt`.

### Usage

```bash
chmod +x mid.sh
./mid.sh
```

## Final Homework

`fin.py` will search `weight1.txt` and output the content of the original document.

